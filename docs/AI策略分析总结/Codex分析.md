# Codex分析

> 基于 2024-2026 A 股环境（T+1、涨跌停、散户主导、政策驱动）对 36 个策略进行投研评估。
>
> 综合评分权重：理论有效性 15% + A股适用性 25% + 信号质量 20% + 风险收益比 20% + 时效性 10% + 可组合性 10%。

## 一、技术面策略（24个）

3. **均线多头排列**（综合 **7.25**）  
   - 六项：理7 / 适8 / 信7 / 风7 / 时6 / 组8

4. **唐奇安通道突破**（综合 **7.50**）

5. **ATR 波动率突破**（综合 **7.50**）

## 二、基本面策略（12个）

28. **市销率低估值**（综合 **5.80**）
