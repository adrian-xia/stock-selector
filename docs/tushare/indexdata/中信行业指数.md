# 中信行业指数

## 接口名称

`ci_daily`

## 接口说明

获取中信行业指数日线行情数据。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 8000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 CI005001.CI） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

**注意：** ts_code 和 trade_date 至少需要填写一个参数。

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| close | float | 收盘点位 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取石油石化行业某日行情
df = pro.ci_daily(ts_code='CI005001.CI', trade_date='20260217')

# 获取石油石化行业一段时间的行情
df = pro.ci_daily(ts_code='CI005001.CI', start_date='20260101', end_date='20260217')

# 获取某日所有中信行业行情
df = pro.ci_daily(trade_date='20260217')
```

## 注意事项

1. 中信行业分类是另一套常用的行业分类标准
2. 中信行业指数包括一级、二级、三级行业
3. 建议按日期批量获取所有行业数据
4. 单次最大提取 8000 行数据
