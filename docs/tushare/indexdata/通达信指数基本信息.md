# 通达信指数基本信息

## 接口名称

`tdx_index`

## 接口说明

获取通达信自定义指数的基本信息，包括指数代码、名称、类别等。

## 接口权限

- 需要 Tushare Pro 积分 >= 120

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 880001.TI） |
| name | str | N | 指数名称 |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| name | str | 指数名称 |
| category | str | 指数类别 |
| market | str | 市场 |
| publisher | str | 发布方 |
| base_date | str | 基期 |
| base_point | float | 基点 |
| list_date | str | 发布日期 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取所有通达信指数基本信息
df = pro.tdx_index()

# 获取特定指数信息
df = pro.tdx_index(ts_code='880001.TI')

# 按名称搜索指数
df = pro.tdx_index(name='新能源')
```

## 返回示例

```
     ts_code      name category market publisher  base_date  base_point  list_date
0  880001.TI    新能源车     概念     通达信      通达信  20100101      1000.0   20100101
1  880002.TI    锂电池      概念     通达信      通达信  20100101      1000.0   20100101
2  880003.TI    光伏概念    概念     通达信      通达信  20100101      1000.0   20100101
```

## 注意事项

1. 通达信指数包含大量细分行业和热门概念指数
2. 指数分类包括行业、概念、地域等多个维度
3. 适合用于行业轮动和概念炒作分析
4. 指数会不定期新增或调整
