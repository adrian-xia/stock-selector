# 指数成分（全量）

## 接口名称

`index_member_all`

## 接口说明

获取指数成分股的全量历史数据，包括历次调整的完整记录。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 10000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 000300.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| index_name | str | 指数名称 |
| con_code | str | 成分股代码 |
| con_name | str | 成分股名称 |
| in_date | str | 纳入日期 |
| out_date | str | 剔除日期 |
| is_new | str | 是否最新成分（Y/N） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取沪深300全量成分股历史
df = pro.index_member_all(ts_code='000300.SH')

# 获取上证50全量成分股历史
df = pro.index_member_all(ts_code='000016.SH')

# 获取某日所有指数成分股
df = pro.index_member_all(trade_date='20260217')
```

## 返回示例

```
  index_code index_name   con_code con_name    in_date  out_date is_new
0  000300.SH     沪深300  600519.SH   贵州茅台  20050408      None      Y
1  000300.SH     沪深300  000858.SZ   五粮液    20050408      None      Y
2  000300.SH     沪深300  000568.SZ   泸州老窖  20100101  20150630      N
```

## 注意事项

1. 该接口需要较高积分（2000+）
2. 返回指数成分股的完整历史调整记录
3. is_new='Y' 表示当前仍在指数中的成分股
4. is_new='N' 表示已被剔除的历史成分股
5. 可用于回测时准确还原历史成分股构成
