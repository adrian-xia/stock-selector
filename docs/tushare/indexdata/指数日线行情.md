# 指数日线行情

## 接口名称

`index_daily`

## 接口说明

获取指数每日行情数据，包括开盘价、最高价、最低价、收盘价、成交量、成交额等。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 8000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 000001.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

**注意：** ts_code 和 trade_date 至少需要填写一个参数。

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| close | float | 收盘点位 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| pre_close | float | 昨收盘点 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取上证指数某日行情
df = pro.index_daily(ts_code='000001.SH', trade_date='20260217')

# 获取上证指数一段时间的行情
df = pro.index_daily(ts_code='000001.SH', start_date='20260101', end_date='20260217')

# 获取某日所有指数行情
df = pro.index_daily(trade_date='20260217')
```

```
1  000300.SH   20260217  4123.45  4110.23  4135.67  4105.89   4118.90   4.55     0.11  18234567.0  21345678.90
```

## 注意事项

1. 单次最大提取 8000 行数据，如需更多数据请分批获取
2. 建议按日期批量获取全市场指数数据，效率更高
3. 成交量单位为手（1手=100股），成交额单位为千元
4. 部分指数可能没有成交量和成交额数据
