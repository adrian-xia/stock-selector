# 指数月线行情

## 接口名称

`index_monthly`

## 接口说明

获取指数月线行情数据。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 4000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | Y | 指数代码（如 000001.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD，每月最后一个交易日） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日（每月最后一个交易日） |
| close | float | 收盘点位 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| pre_close | float | 上月收盘点 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取上证指数月线行情
df = pro.index_monthly(ts_code='000001.SH', start_date='20240101', end_date='20260217')

# 获取沪深300月线行情
df = pro.index_monthly(ts_code='000300.SH', start_date='20240101', end_date='20260217')
```

```
     ts_code trade_date     close      open      high       low  pre_close  change  pct_chg          vol         amount
0  000001.SH   20260131  3198.50  3050.45  3256.78  3045.34   3089.20  109.30     3.54  2567890000.0  2845678900.00
1  000001.SH   20251231  3089.20  2980.23  3115.67  2975.89   2965.50  123.70     4.17  2456789000.0  2734567890.00
```

## 注意事项

1. trade_date 为每月最后一个交易日
2. 月线数据由日线数据聚合而成
3. 成交量和成交额为该月累计值
4. 单次最大提取 4000 行数据
