# 全球指数

## 接口名称

`index_global`

## 接口说明

获取全球主要市场指数的日线行情数据，包括美股、港股、欧洲、亚太等主要指数。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 2000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 DJI.GI） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| close | float | 收盘点位 |
| pre_close | float | 昨收点位 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量 |
| amount | float | 成交额 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取道琼斯指数某日行情
df = pro.index_global(ts_code='DJI.GI', trade_date='20260217')

# 获取纳斯达克指数一段时间的行情
df = pro.index_global(ts_code='IXIC.GI', start_date='20260101', end_date='20260217')

# 获取恒生指数行情
df = pro.index_global(ts_code='HSI.GI', start_date='20260101', end_date='20260217')

# 获取某日所有全球指数行情
df = pro.index_global(trade_date='20260217')
```

## 注意事项

1. 包含全球主要市场指数：美股（道琼斯、纳斯达克、标普500）、港股（恒生指数）、欧洲（富时100、德国DAX）、亚太等
2. 不同市场的交易时间不同，需注意时区差异
3. 成交量和成交额单位因市场而异
4. 单次最大提取 2000 行数据
