# 指数每日指标

## 接口名称

`index_dailybasic`

## 接口说明

获取指数每日技术指标数据，包括换手率、市盈率、市净率等。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 000001.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

**注意：** ts_code 和 trade_date 至少需要填写一个参数。

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| total_mv | float | 总市值（亿元） |
| float_mv | float | 流通市值（亿元） |
| total_share | float | 总股本（亿股） |
| float_share | float | 流通股本（亿股） |
| free_share | float | 自由流通股本（亿股） |
| turnover_rate | float | 换手率（%） |
| turnover_rate_f | float | 换手率（自由流通股本）（%） |
| pe | float | 市盈率 |
| pe_ttm | float | 市盈率TTM |
| pb | float | 市净率 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取上证指数某日指标
df = pro.index_dailybasic(ts_code='000001.SH', trade_date='20260217')

# 获取上证指数一段时间的指标
df = pro.index_dailybasic(ts_code='000001.SH', start_date='20260101', end_date='20260217')

# 获取某日所有指数指标
df = pro.index_dailybasic(trade_date='20260217')
```

## 注意事项

1. 该接口提供指数层面的估值指标
2. 市盈率、市净率为成分股加权平均值
3. 换手率基于流通股本或自由流通股本计算
4. 部分指数可能缺少某些指标数据
