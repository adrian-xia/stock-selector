# 通达信指数行情

## 接口名称

`tdx_daily`

## 接口说明

获取通达信自定义指数日线行情数据。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 8000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 880001.TI） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

**注意：** ts_code 和 trade_date 至少需要填写一个参数。

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| close | float | 收盘点位 |
| open | float | 开盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| pre_close | float | 昨收点位 |
| change | float | 涨跌点 |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取通达信指数某日行情
df = pro.tdx_daily(ts_code='880001.TI', trade_date='20260217')

# 获取通达信指数一段时间的行情
df = pro.tdx_daily(ts_code='880001.TI', start_date='20260101', end_date='20260217')

# 获取某日所有通达信指数行情
df = pro.tdx_daily(trade_date='20260217')
```

## 注意事项

1. 通达信指数是通达信软件自定义的行业/概念指数
2. 包含大量细分行业和热门概念指数
3. 适合用于行业轮动和概念炒作分析
4. 单次最大提取 8000 行数据
