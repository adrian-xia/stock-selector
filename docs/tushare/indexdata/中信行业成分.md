# 中信行业成分

## 接口名称

`ci_index_member`

## 接口说明

获取中信行业成分股信息，包括各行业包含的股票列表。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| index_code | str | Y | 指数代码（如 CI005001.CI） |
| ts_code | str | N | 股票代码（如 600519.SH） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| index_name | str | 指数名称 |
| con_code | str | 成分股代码 |
| con_name | str | 成分股名称 |
| in_date | str | 纳入日期 |
| out_date | str | 剔除日期 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取石油石化行业成分股
df = pro.ci_index_member(index_code='CI005001.CI')

# 获取煤炭行业成分股
df = pro.ci_index_member(index_code='CI005002.CI')

# 查询某股票所属中信行业
df = pro.ci_index_member(ts_code='600519.SH')
```

## 返回示例

```
    index_code index_name   con_code con_name    in_date out_date
0  CI005001.CI     石油石化  600519.SH   贵州茅台  20100101     None
1  CI005001.CI     石油石化  000858.SZ   五粮液    20100101     None
2  CI005001.CI     石油石化  000568.SZ   泸州老窖  20100101     None
```

## 注意事项

1. 该接口需要较高积分（2000+）
2. 中信行业分类与申万行业分类是两套不同的标准
3. 行业成分股会定期调整，需关注 in_date 和 out_date
4. 单次最大提取 5000 行，如需更多数据请分批获取
