# 申万行业分类

## 接口名称

`index_classify`

## 接口说明

获取申万行业分类信息，包括一级、二级、三级行业分类。

## 接口权限

- 需要 Tushare Pro 积分 >= 120

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| index_code | str | N | 指数代码（如 801010.SI） |
| level | str | N | 行业分级（L1-一级，L2-二级，L3-三级） |
| src | str | N | 指数来源（SW-申万） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| industry_name | str | 行业名称 |
| level | str | 行业级别（L1/L2/L3） |
| industry_code | str | 行业代码 |
| src | str | 指数来源 |
| parent_code | str | 父级代码 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取申万一级行业分类
df = pro.index_classify(level='L1', src='SW')

# 获取申万二级行业分类
df = pro.index_classify(level='L2', src='SW')

# 获取申万三级行业分类
df = pro.index_classify(level='L3', src='SW')

# 获取特定行业信息
df = pro.index_classify(index_code='801010.SI')
```

## 返回示例

```
  index_code industry_name level industry_code src parent_code
0  801010.SI          农林牧渔    L1        801010  SW        None
1  801020.SI          采掘      L1        801020  SW        None
2  801030.SI          化工      L1        801030  SW        None
3  801040.SI          钢铁      L1        801040  SW        None
```

## 注意事项

1. 申万行业分类是A股市场最常用的行业分类标准
2. 一级行业共 31 个，二级行业共 104 个，三级行业共 227 个
3. 行业分类会定期调整，需关注更新时间
4. parent_code 字段用于建立行业层级关系
