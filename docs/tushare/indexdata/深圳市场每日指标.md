# 深圳市场每日指标

## 接口名称

`sz_daily_info`

## 接口说明

获取深圳市场每日交易统计指标，包括主板、中小板、创业板的分类统计数据。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| trade_date | str | 交易日期 |
| ts_code | str | 市场代码 |
| ts_name | str | 市场名称 |
| com_count | int | 上市公司数量 |
| total_share | float | 总股本（亿股） |
| float_share | float | 流通股本（亿股） |
| total_mv | float | 总市值（亿元） |
| float_mv | float | 流通市值（亿元） |
| amount | float | 成交额（亿元） |
| vol | float | 成交量（亿股） |
| deal_amount | float | 成交笔数（万笔） |
| pe | float | 平均市盈率 |
| tr | float | 换手率（%） |
| up_count | int | 上涨家数 |
| down_count | int | 下跌家数 |
| limit_up_count | int | 涨停家数 |
| limit_down_count | int | 跌停家数 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取深圳市场某日指标
df = pro.sz_daily_info(trade_date='20260217')

# 获取深圳市场一段时间的指标
df = pro.sz_daily_info(start_date='20260101', end_date='20260217')
```

## 注意事项

1. 该接口提供深圳市场的分板块统计数据
2. 包括主板、中小板、创业板的独立统计
3. 可用于分析不同板块的市场表现差异
4. 数据每日更新，建议按日期批量获取
