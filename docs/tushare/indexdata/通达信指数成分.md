# 通达信指数成分

## 接口名称

`tdx_member`

## 接口说明

获取通达信自定义指数的成分股信息。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| index_code | str | Y | 指数代码（如 880001.TI） |
| ts_code | str | N | 股票代码（如 600519.SH） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| index_name | str | 指数名称 |
| con_code | str | 成分股代码 |
| con_name | str | 成分股名称 |
| in_date | str | 纳入日期 |
| out_date | str | 剔除日期 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取新能源车指数成分股
df = pro.tdx_member(index_code='880001.TI')

# 获取锂电池指数成分股
df = pro.tdx_member(index_code='880002.TI')

# 查询某股票所属通达信指数
df = pro.tdx_member(ts_code='600519.SH')
```

## 返回示例

```
  index_code index_name   con_code con_name    in_date out_date
0  880001.TI    新能源车  600519.SH   贵州茅台  20200101     None
1  880001.TI    新能源车  000858.SZ   五粮液    20200101     None
2  880001.TI    新能源车  000568.SZ   泸州老窖  20200101  20220630
```

## 注意事项

1. 该接口需要较高积分（2000+）
2. 通达信指数成分股会根据市场热点动态调整
3. 适合用于捕捉市场热点和概念炒作
4. 单次最大提取 5000 行，如需更多数据请分批获取
