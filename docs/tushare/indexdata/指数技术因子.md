# 指数技术因子

## 接口名称

`index_factor_pro`

## 接口说明

获取指数技术因子数据，包括动量、波动率、流动性等多维度技术指标。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 000300.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| trade_date | str | 交易日 |
| momentum_1m | float | 1个月动量 |
| momentum_3m | float | 3个月动量 |
| momentum_6m | float | 6个月动量 |
| momentum_12m | float | 12个月动量 |
| volatility_1m | float | 1个月波动率 |
| volatility_3m | float | 3个月波动率 |
| volatility_6m | float | 6个月波动率 |
| volatility_12m | float | 12个月波动率 |
| liquidity | float | 流动性指标 |
| turnover_rate | float | 换手率 |
| amplitude | float | 振幅 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取沪深300某日技术因子
df = pro.index_factor_pro(ts_code='000300.SH', trade_date='20260217')

# 获取沪深300一段时间的技术因子
df = pro.index_factor_pro(ts_code='000300.SH', start_date='20260101', end_date='20260217')

# 获取某日所有指数技术因子
df = pro.index_factor_pro(trade_date='20260217')
```

## 注意事项

1. 该接口需要较高积分（2000+）
2. 提供多时间周期的动量和波动率指标
3. 适合用于量化策略开发和因子分析
4. 数据每日更新，建议按日期批量获取
