# 大盘指数每日指标

## 接口名称

`daily_info`

## 接口说明

获取大盘（上证/深证）每日交易统计指标，包括涨跌家数、涨停跌停数、总成交额等市场宽度指标。

## 接口权限

- 需要 Tushare Pro 积分 >= 120
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |
| ts_code | str | N | 市场代码（SSE-上交所，SZSE-深交所） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| trade_date | str | 交易日期 |
| ts_code | str | 市场代码 |
| ts_name | str | 市场名称 |
| com_count | int | 上市公司数量 |
| total_share | float | 总股本（亿股） |
| float_share | float | 流通股本（亿股） |
| total_mv | float | 总市值（亿元） |
| float_mv | float | 流通市值（亿元） |
| amount | float | 成交额（亿元） |
| vol | float | 成交量（亿股） |
| trans_count | int | 成交笔数（万笔） |
| pe | float | 平均市盈率 |
| tr | float | 换手率（%） |
| exchange_rate | float | 量比 |
| up_count | int | 上涨家数 |
| down_count | int | 下跌家数 |
| limit_up_count | int | 涨停家数 |
| limit_down_count | int | 跌停家数 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取上交所某日市场指标
df = pro.daily_info(trade_date='20260217', ts_code='SSE')

# 获取深交所某日市场指标
df = pro.daily_info(trade_date='20260217', ts_code='SZSE')

# 获取上交所一段时间的市场指标
df = pro.daily_info(start_date='20260101', end_date='20260217', ts_code='SSE')

# 获取某日所有市场指标
df = pro.daily_info(trade_date='20260217')
```

## 注意事项

1. 该接口提供市场宽度指标，用于判断市场整体强弱
2. 涨跌家数、涨停跌停数是重要的市场情绪指标
3. 总成交额、换手率反映市场活跃度
4. 数据每日更新，建议按日期批量获取
