# 申万行业成分

## 接口名称

`index_member`

## 接口说明

获取申万行业成分股信息，包括各行业包含的股票列表。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| index_code | str | Y | 指数代码（如 801010.SI） |
| ts_code | str | N | 股票代码（如 600519.SH） |
| is_new | str | N | 是否最新（Y-是，N-否） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| index_name | str | 指数名称 |
| con_code | str | 成分股代码 |
| con_name | str | 成分股名称 |
| in_date | str | 纳入日期 |
| out_date | str | 剔除日期 |
| is_new | str | 是否最新成分（Y/N） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取农林牧渔行业最新成分股
df = pro.index_member(index_code='801010.SI', is_new='Y')

# 获取化工行业最新成分股
df = pro.index_member(index_code='801030.SI', is_new='Y')

# 查询某股票所属行业
df = pro.index_member(ts_code='600519.SH', is_new='Y')
```

## 返回示例

```
  index_code index_name   con_code con_name    in_date out_date is_new
0  801010.SI     农林牧渔  600519.SH   贵州茅台  20100101     None      Y
1  801010.SI     农林牧渔  000858.SZ   五粮液    20100101     None      Y
2  801010.SI     农林牧渔  000568.SZ   泸州老窖  20100101     None      Y
```

## 注意事项

1. 该接口需要较高积分（2000+）
2. is_new='Y' 可获取最新成分股列表
3. 行业成分股会定期调整，需关注 in_date 和 out_date
4. 单次最大提取 5000 行，如需更多数据请分批获取
