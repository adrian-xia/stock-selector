# 指数成分和权重

## 接口名称

`index_weight`

## 接口说明

获取各类指数成分股及权重数据，包括沪深300、中证500等主要指数的成分股信息。

## 接口权限

- 需要 Tushare Pro 积分 >= 2000
- 单次最大提取 5000 行数据

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| index_code | str | Y | 指数代码（如 000300.SH） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| index_code | str | 指数代码 |
| con_code | str | 成分股代码 |
| trade_date | str | 交易日期 |
| weight | float | 权重（%） |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取沪深300最新成分股及权重
df = pro.index_weight(index_code='000300.SH')

# 获取沪深300某日成分股及权重
df = pro.index_weight(index_code='000300.SH', trade_date='20260217')

# 获取沪深300一段时间的成分股变化
df = pro.index_weight(index_code='000300.SH', start_date='20260101', end_date='20260217')
```

## 返回示例

```
  index_code   con_code trade_date  weight
0  000300.SH  600519.SH   20260217    3.45
1  000300.SH  601318.SH   20260217    2.89
2  000300.SH  600036.SH   20260217    2.34
3  000300.SH  000858.SZ   20260217    1.98
```

## 注意事项

1. 该接口需要较高积分（2000+），主要用于量化策略开发
2. 权重数据通常每季度调整一次（3月、6月、9月、12月）
3. 权重总和应为 100%，可用于验证数据完整性
4. 部分指数可能不提供权重数据，只提供成分股列表
5. 单次最大提取 5000 行，如需更多数据请分批获取
