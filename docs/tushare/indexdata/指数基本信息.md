# 指数基本信息

## 接口名称

`index_basic`

## 接口说明

获取指数基础信息，包括指数代码、名称、类别、发布方等。

## 接口权限

- 需要 Tushare Pro 积分 >= 120

## 接口参数

| 参数名 | 类型 | 必选 | 说明 |
|:---|:---|:---|:---|
| ts_code | str | N | 指数代码（如 000001.SH） |
| name | str | N | 指数简称 |
| market | str | N | 交易所或服务商（SSE-上交所，SZSE-深交所，CICC-中金所，SW-申万，OTH-其他） |
| publisher | str | N | 发布商 |
| category | str | N | 指数类别 |

## 返回字段

| 字段名 | 类型 | 说明 |
|:---|:---|:---|
| ts_code | str | TS代码 |
| name | str | 简称 |
| fullname | str | 指数全称 |
| market | str | 市场 |
| publisher | str | 发布方 |
| index_type | str | 指数风格 |
| category | str | 指数类别 |
| base_date | str | 基期 |
| base_point | float | 基点 |
| list_date | str | 发布日期 |
| weight_rule | str | 加权方式 |
| desc | str | 描述 |
| exp_date | str | 终止日期 |

## 使用示例

```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取所有指数
df = pro.index_basic()

# 获取上交所指数
df = pro.index_basic(market='SSE')

# 获取沪深300指数信息
df = pro.index_basic(ts_code='000300.SH')
```

## 返回示例

```
     ts_code      name  ...  weight_rule                                               desc
0  000001.SH      上证指数  ...         流通股本  反映上海证券交易所上市股票价格的变动情况
1  000300.SH    沪深300  ...         流通股本  由沪深A股中规模大、流动性好的最具代表性的300只股票组成
2  399001.SZ      深证成指  ...         流通股本  反映深圳证券市场的运行状况
3  399006.SZ    创业板指  ...         流通股本  反映创业板市场层次的运行情况
```

## 注意事项

1. 该接口返回所有指数的基本信息，包括主要指数、行业指数、概念指数等
2. 可通过 market 参数筛选不同交易所的指数
3. 申万行业指数的 market 为 'SW'
4. 部分指数可能已终止，需关注 exp_date 字段
