# 量价关系八大口诀

> 来源：网络文章《炒股有天才吗？》

## 口诀一：放量上涨，必然回调

放量上涨必然回调。短线连续放量上涨，多头力量释放过强，短线必有回调需求，表现在 K 线上就是**连续上涨后必有阴跌**。

```
        ┌── 上涨
       /
      / ← 放量上涨，多头力量释放过强，
     /    短线必会有空头进攻
    /
───┘
   放量
```

## 口诀二：放量下跌，必会回调

在下跌趋势中，突然放量加速下跌，同样说明空头力量释放过强，多头要反击，表现在 K 线上就是**加速下跌出反弹**。

```
───┐
    \
     \ ← 下跌趋势，突然放量加速下跌，
      \   短线多头必会反攻
       \
        └── 短线必会反弹
   放量
```

## 口诀三：缩量上涨，还会上涨

缩量上涨，还会上涨。上涨但是量能平稳，说明没有新资金介入，**主力正在默默地抬升股价**。等到放量说明主力即将出货。

```
        ┌── 稳定上涨
       /
      / ← 缩量上涨代表主力正在拉升，
     /    而且没有出货的迹象，要立刻进场
    /
───┘
   缩量
```

## 口诀四：缩量下跌，还会下跌

缩量下跌，还会下跌。下跌无量说明大家一致看跌，有分歧才会有成交量，所以**缩量代表当前的趋势短期不会改变**。

```
───┐
    \
     \ ← 连续下跌缩量代表市场一致看空，
      \   处于一边倒的趋势
       \
        └── 连续下跌
   缩量
```

## 口诀五：放量下跌，头部已现

放量下跌，头部已现。成交量代表着分歧，既然放量说明当天分歧很大，但是收了大阴线，说明**空头力量碾压多头，顶部形成**。

```
───┐
    \ ← 突然收巨量阴线
     \
      └── 顶部形成
   放量
```

## 口诀六：天量顶下跌，后市会大涨

天量顶下跌，后市会大涨。股价接连放天量，但是下跌幅度有限，**雷声大雨点小，说明主力在震仓**，后市会大涨创新高。

```
    小阴线  小阴线
───┤    ├──┤    ├─── 大涨 ──→
   突然放天量（但下跌幅度有限）
```

## 口诀七：放量顶下跌，后市调整长

放量顶下跌，后市调整长。顶部下跌伴随巨量阴线，说明**主力已经出货**，并且短时间不会再来吸筹拉升。

```
───┐
    \ ← 放量伴随大阴线
     \
      \── 后市调整长
   放量（主力已出货）
```

## 口诀八：后量超前量，后市要跟上

后量超前量，后市要跟上。成交量接连创新高代表**资金在不断进场**，那行情短期不会结束，大概率接连上涨。

```
        ┌── 上涨
       /
      / ← 成交量接连创新高，
     /    代表资金在不断地流入
    /
───┘
   后量超前量
```
